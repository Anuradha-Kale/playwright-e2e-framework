# Playwright E2E Automation Framework

## 📌 Overview
This project is an end-to-end test automation framework built using Playwright with Page Object Model (POM) design pattern.

## 🚀 Features
- Page Object Model (POM)
- Data-driven testing using JSON
- End-to-End scenarios (Login, Cart, Checkout)
- HTML Reporting
- Screenshot, video & trace on failure
- GitHub Actions CI integration

## 🛠️ Tech Stack
- Playwright
- JavaScript
- Node.js

## ▶️ Run Tests
npx playwright test

## 📊 View Report
npx playwright show-report
